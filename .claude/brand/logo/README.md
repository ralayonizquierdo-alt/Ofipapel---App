# Logotipo oficial

- `logo-oficial-ofipapel.png` — logotipo oficial aprobado por el
  propietario (fondo transparente, versión a color).

Reglas de uso en `../reglas.md` (sección 1). Resumen: usar siempre este
archivo tal cual, nunca recrearlo con IA, y colocarlo siempre sobre una
zona blanca.

Si se necesita una variante nueva (monocromo, sobre fondo oscuro, otro
formato de exportación...), debe generarse a partir de este mismo archivo
original y aprobarse por el propietario antes de añadirse aquí — nunca
redibujada desde cero.

El fondo de marca del que sale la paleta oficial de verdes está en
`../colores/fondo-marca-origen-paleta.jpg` (ver `../colores/paleta.md`).
