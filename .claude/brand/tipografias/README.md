# Tipografías oficiales — OFIPAPEL

Verificadas contra `Index.html` (documentadas también en
`design-studio/README.md`, sección de brand kit). No inventadas.

- **Inter** — texto general. Vía Google Fonts `@import` (así es como ya se
  carga en `Index.html`). No disponible en el plan de Adobe Fonts
  conectado (`not_entitled`) — para piezas destinadas a Adobe Express, si
  no está disponible usar Google Fonts `@import` para no perder la
  fidelidad de marca.
- **IBM Plex Mono** — etiquetas, datos, acentos técnicos (p. ej. precios,
  referencias). Vía Google Fonts `@import`, y disponible también en Adobe
  Fonts (`IBMPlexMono-Bold`).

Ningún archivo de fuente se versiona aquí (ambas se cargan vía Google
Fonts/Adobe Fonts). Si en el futuro se aprueban fuentes adicionales o
variantes, se documentan en este archivo.
