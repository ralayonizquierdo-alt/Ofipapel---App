# Plantillas aprobadas

Plantillas de diseño (banners, posts, flyers, carteles...) ya aprobadas
por el propietario como referencia oficial de estilo, distintas de los
borradores de trabajo de `design-studio/templates/`.

Vacío por ahora. Cuando el propietario apruebe una plantilla concreta como
"así es como debe ser", se propondrá copiarla aquí.
