# Prompts de generación aprobados

Prompts de generación de imagen (p. ej. para Adobe Firefly, vía
`design-studio/scripts/firefly-generate.js`) ya aprobados por el
propietario para escenas de producto: cambios de fondo, iluminación y
composición sin alterar el producto original (ver regla 2 en
`../reglas.md`).

Vacío por ahora. Cuando el propietario apruebe un prompt concreto como
fiable para repetir, se propondrá guardarlo aquí.
